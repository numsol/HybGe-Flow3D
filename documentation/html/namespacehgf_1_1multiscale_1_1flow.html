hgf::multiscale::flow::compute_permeability_tensor computes upscaled permeability tensor given flow solutions for flows in each principal axis direction.

Parameters
[in]par- parameters struct containing problem information.
[in]pressure_ib_list- Integer array indicating if a cell is an immersed boundary cell.
[in]velocity_u- Degrees of freedom for the x-component of the porescale velocity solution.
[in]velocity_v- Degrees of freedom for the y-component of the porescale velocity solution.
[in]velocity_w- Degrees of freedom for the z-component of the porescale velocity solution. Can be empty if par indicates a 2d flow.
[in]solution_xflow- Solution of flow problem in x-axis direction.
[in]solution_yflow- Solution of flow problem in y-axis direction.
[in]solution_zflow- Solution of flow problem in z-axis direction. Can be empty if par indicates a 2d flow.
[out]permeability- resulting 4x4 (2d flow) or 9x9 (3d flow) permeability tensor.
